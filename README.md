# Inkthar Releases

This repository hosts **binary releases** for the Inkthar desktop application.

Download the latest version from the [Releases](https://github.com/wilsonrljr/inkthar-releases/releases) page.

| Platform | File |
|----------|------|
| macOS (Universal) | `Inkthar-<version>-mac.dmg` |
| Windows (x64) | `Inkthar-<version>-windows.exe` |

## Verification

Each release includes a `SHA256SUMS.txt` file. To verify a download:

```bash
shasum -a 256 -c SHA256SUMS.txt
```
